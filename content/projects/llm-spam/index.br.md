O LLM Spam foi um breve estudo sobre como os LLMs irão impactar a internet como a conhecemos. O experimento consistiu em desenvolver um script em Python que utilizaria a API do ChatGPT da OpenAI para criar artigos. O objetivo era determinar se seria possível fazer isso em larga escala, usar táticas de clickbait e manipular usuários para que interagissem com os artigos. A resposta é **sim**.

Este foi um projeto extremamente interessante que não me ensinou muito sobre programação, mas foi muito provocativo em relação a como a internet poderá ser no futuro e os perigos dos LLMs.

Alguns fatos interessantes:

- O projeto inteiro custou menos de $25 em créditos de API
- 1.082 posts foram gerados e publicados automaticamente ao longo de alguns dias
- 1,32M palavras foram escritas
- O site recebeu 2.712 visitantes e 3.334 visualizações
- Posts que provocaram raiva nos visitantes, geralmente quando continham desinformação devido a alucinações do ChatGPT, foram os que tiveram melhor desempenho

O código deste projeto não é open source, pois acredito que sua publicação seria prejudicial para a saúde da internet.
