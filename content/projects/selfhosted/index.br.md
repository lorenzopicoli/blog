Este projeto contém todo o código necessário para replicar completamente minha configuração de homelab do zero. Ele inclui todos os playbooks Ansible necessários para configurar hosts e serviços desde o início.

Este projeto não existe em um site, mas é um esforço contínuo para permitir que qualquer pessoa (incluindo eu mesmo) replique minha configuração de homelab, iniciando todas as máquinas host e contêineres necessários.

Para saber mais sobre este projeto, confira meu [resumo de fim de ano de 2024](/posts/2024-selfhosted-recap/).

**Confira esse projeto no [GitHub](https://github.com/lorenzopicoli/selfhosted)**
